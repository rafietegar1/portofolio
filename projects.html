Project Bimkar-Hospital

Bimkar-Hospital is a simple hospital appointment system developed using the Laravel framework, focusing on interaction between doctors (dosen) and patients. This project allows patients to register, book appointments, and view their appointment history, while doctors can manage their schedules and view patient data.

In this project, I worked on features such as user authentication (for both doctors and patients), appointment booking system, patient dashboard, and role-based access control. The interface was built using Laravel Blade and styled with Bootstrap for a clean and responsive layout.

This project helped me strengthen my skills in Laravel development, especially in building relational databases, authentication systems, and dynamic web interfaces.

Sipanjir - Website

SIPANJIR (Sistem Informasi Pemantauan Banjir) is a web-based application designed to monitor flood risks based on rainfall data and news reports. The system aims to help communities and local authorities stay informed about potential flooding events.

In this project, I developed features such as a dashboard for real-time rainfall data (integrated via API), a flood history page based on curated news articles, and an educational section to raise awareness about flood prevention and preparedness. The website was built using modern web technologies with a focus on usability and accessibility.

This project enhanced my experience in API integration, data visualization, and building user-friendly interfaces for public information systems.

Hospital Appointment System

This is a web-based hospital appointment system built using Laravel. The system enables patients to book appointments with doctors, view their appointment history, and manage their profiles. Doctors can view their patient queue, record examination results, and manage appointment schedules.

I developed core features such as multi-role authentication (admin, doctor, patient), patient registration, appointment management, and medical record history. The project implements a responsive user interface using Bootstrap and Laravel Blade.

This project gave me deeper experience in role-based access control, form validation, database relationships, and full-stack web development using Laravel.
